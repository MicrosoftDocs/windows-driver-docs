# Obtaining Information about a Parallel Device





In addition to [connecting to a parallel device](connecting-to-a-parallel-device.md), a client can use the following device control requests to obtain additional information about a parallel device:

[**IOCTL\_IEEE1284\_GET\_MODE**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_ieee1284_get_mode)

[**IOCTL\_PAR\_GET\_DEFAULT\_MODES**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_get_default_modes)

[**IOCTL\_PAR\_GET\_DEVICE\_CAPS**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_get_device_caps)

[**IOCTL\_PAR\_IS\_PORT\_FREE**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_is_port_free)

[**IOCTL\_PAR\_QUERY\_DEVICE\_ID\_SIZE**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_query_device_id_size)

[**IOCTL\_PAR\_QUERY\_DEVICE\_ID**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_query_device_id)

[**IOCTL\_PAR\_QUERY\_INFORMATION**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_query_information)

[**IOCTL\_PAR\_QUERY\_LOCATION**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_query_location)

[**IOCTL\_PAR\_QUERY\_RAW\_DEVICE\_ID**](/windows-hardware/drivers/ddi/ntddpar/ni-ntddpar-ioctl_par_query_raw_device_id)

[**IOCTL\_SERIAL\_GET\_TIMEOUTS**](/windows-hardware/drivers/ddi/ntddser/ni-ntddser-ioctl_serial_get_timeouts)
