## Other WDK Downloads

This page contains previous version of WDK that Microsoft has now unpublished. We no longer maintain these kits and they are no longer supported. All driver developer should use the latest WDK release for all Windows driver projects. If you believe you have valid justiication to use a WDK that is no longer available, please complete this [form](https://forms.office.com/Pages/ResponsePage.aspx?id=v4j5cvGGr0GRqy180BHbR0aScSwEgKxJvzxab6T-IN5UNFlHSkFQTjBLS1lRS08wTFRZME0yRVRVVi4u) and submit to WDK feedback@microsoft.com
