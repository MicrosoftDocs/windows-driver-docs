# !ppmperf

The **!ppmperf** extension command displays power and performance constraint information.

```dbgcmd
!ppmperf
```

## Parameters

None

## DLL

kedexts.dll

**NOTE:** This command is not supported on all versions of Windows, such as Windows 10, Version 1703.
