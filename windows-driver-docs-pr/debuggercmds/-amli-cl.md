# !amli cl

The **!amli cl** extension clears the AML interpreter's event log.

Syntax

```dbgcmd
    !amli cl
```

## DLL

Kdexts.dll

## Additional Information

For information about related commands and their uses, see [The AMLI Debugger](../debugger/the-amli-debugger.md).
