# !kb, !kv

The !kb and !kv extension commands are obsolete. Use the [**kb (Display Stack Backtrace)**](k--kb--kc--kd--kp--kp--kv--display-stack-backtrace-.md) and **kv (Display Stack Backtrace)** commands instead.
