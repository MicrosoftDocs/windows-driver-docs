# !mtrr

The **!mtrr** extension displays the contents of the MTRR register.

```dbgcmd
!mtrr
```

## <span id="ddk__mtrr_dbg"></span><span id="DDK__MTRR_DBG"></span>


### DLL

Kdexts.dll

 

This extension command can only be used with an x86-based target computer.
