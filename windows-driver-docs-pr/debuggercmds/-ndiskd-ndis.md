# !ndiskd.ndis


The **!ndiskd.ndis** extension displays build information about ndis.sys.

```console
!ndiskd.ndis 
```

## Parameters


This extension has no parameters.

## DLL

Ndiskd.dll

## Examples

The following example shows that the debugee machine has a Checked Ndis build.

```console
0: kd> !ndiskd.ndis
Checked Ndis
```
