# !usb3kd.usbanalyze

The [**!usb3kd.usbanalyze**](-usb3kd-device-info.md) extension analyzes a USB 3.0 bug check.

```dbgcmd
!usb3kd.usbanalyze [-v]
```

## Parameters

<span id="_______-v______"></span><span id="_______-V______"></span> **-v**   
The display is verbose.

## DLL

Usb3kd.dll
