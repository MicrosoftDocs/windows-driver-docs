# !ppmperfpolicy

The **!ppmperfpolicy** extension command is no longer supported, use the [!ppmsettings](-ppmsettings.md) command instead.

```dbgcmd
!ppmperfpolicy
```

## Parameters

None

## DLL

Kdexts.dll
