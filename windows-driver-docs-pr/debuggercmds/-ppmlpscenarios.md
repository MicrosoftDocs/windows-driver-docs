# !ppmlpscenarios

The **!ppmlpscenarios** extension command displays per-scenario policy overrides.

```dbgcmd
!ppmlpscenarios
```

## Parameters

None

This command is supported on the following versions of Windows:

- Windows 8

## DLL

Kdexts.dll
