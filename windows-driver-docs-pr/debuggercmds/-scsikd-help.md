# !scsikd.help

The **!scsikd.help** extension displays help text for Scsikd.dll extension commands.

```dbgcmd
!scsikd.help 
```

## DLL

Scsikd.dll

## Additional Information

For more information, see [SCSI Miniport Debugging](../debugger/scsi-miniport-debugging.md).
