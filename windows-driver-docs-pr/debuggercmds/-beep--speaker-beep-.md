# .beep (Speaker Beep)


The **.beep** command makes noise on the computer speaker.

```dbgcmd
    .beep
```

### Environment

This command cannot be used in script files.

|  Item  | Description          |
|--------|----------------------|
|Modes   |User mode, kernel mode|
|Targets |Live, crash dump      |
|Platforms|All                  |
