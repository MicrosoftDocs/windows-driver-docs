# !ppmidle

The **!ppmidle** extension command displays processor idle states.

```dbgcmd
!ppmidle
```

## Parameters

None

This command is supported on the following versions of Windows:

- Windows 7
- Windows 8
- Windows 8.1 
- Windows 10, Version 1511
- Windows 10, Version 1607
- Windows 10, Version 1703

## DLL

Kdexts.dll
