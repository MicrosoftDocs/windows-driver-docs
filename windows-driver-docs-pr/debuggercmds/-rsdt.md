# !rsdt

The **!rsdt** extension displays the ACPI Root System Description Table.

Syntax

```dbgcmd
!rsdt 
```

## DLL

Kdexts.dll

## Additional Information

For more information, see [ACPI Debugging](../debugger/acpi-debugging.md).
