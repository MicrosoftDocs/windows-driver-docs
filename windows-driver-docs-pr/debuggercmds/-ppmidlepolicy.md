# !ppmidlepolicy

The **!ppmidlepolicy** command displays C-state policy.

```dbgcmd
!ppmidlepolicy
```

## Parameters

None

This command is supported on the following versions of Windows:

- Windows 7

## DLL

Kdexts.dll
