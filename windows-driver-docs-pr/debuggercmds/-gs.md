# !gs

The **!gs** extension analyzes a /GS stack overflow.

```dbgcmd
!gs
```

## DLL

Ext.dll

## Remarks

The **!gs**extension helps debug buffer overruns. Run **!gs**when you encounter a STATUS\_STACK\_BUFFER\_OVERRUN error, as the following example shows.
