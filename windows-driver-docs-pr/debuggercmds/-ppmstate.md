# !ppmstate

The **!ppmstate** extension command displays power state information.

```dbgcmd
!ppmstate
```

## Parameters

None

## DLL

Kdexts.dll

**NOTE:** This command is not supported on all versions of Windows, such as Windows 10, Version 1703.
