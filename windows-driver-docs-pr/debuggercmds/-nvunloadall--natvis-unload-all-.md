# .nvunloadall (NatVis Unload All)

The .nvunloadall command unloads all of the NatVis files from the debugger environment.

```dbgcmd
 .nvunloadall
```

## Environment

|  Item  | Description          |
|--------|----------------------|
|Modes   |User mode, kernel mode|
|Targets |Live, crash dump      |
|Platforms|All                  |
