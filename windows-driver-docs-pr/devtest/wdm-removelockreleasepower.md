# RemoveLockReleasePower rule (wdm)


The **RemoveLockReleasePower** rule verifies that calls to [**IoAcquireRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff548204) and [**IoReleaseRemoveLock**](https://msdn.microsoft.com/library/windows/hardware/ff549560) are used in strict alternation. Moreover, at the end of the dispatch routine the driver should not hold the remove lock.

|              |     |
|--------------|-----|
| Driver model | WDM |
