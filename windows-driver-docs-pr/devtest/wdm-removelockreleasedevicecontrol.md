# RemoveLockReleaseDeviceControl rule (wdm)


The **RemoveLockReleaseDeviceControl** rule verifies that calls to [**IoAcquireRemoveLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-ioacquireremovelock) and [**IoReleaseRemoveLock**](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-ioreleaseremovelock) are used in strict alternation. Moreover, at the end of the dispatch routine the driver should not hold the remove lock.

|              |     |
|--------------|-----|
| Driver model | WDM |
