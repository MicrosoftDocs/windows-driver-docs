# RemoveLockReleaseShutdown rule (wdm)


The **RemoveLockReleaseShutdown** rule verifies that calls to [**IoAcquireRemoveLock**](/windows-hardware/drivers/ddi/wdm/nf-wdm-ioacquireremovelock) and [**IoReleaseRemoveLock**](/windows-hardware/drivers/ddi/wdm/nf-wdm-ioreleaseremovelock) are used in strict alternation. Moreover, at the end of the dispatch routine the driver should not hold the remove lock.

**Driver model: WDM**
