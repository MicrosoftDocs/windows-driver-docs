# !bthkd.rfcomminfo


The **!bthkd.rfcomminfo** command displays information about the RFCOMM FDO and the TDI Device Object.

```dbgsyntax
!bthkd.rfcomminfo
```

## <span id="DLL"></span><span id="dll"></span>DLL


Bthkd.dll
