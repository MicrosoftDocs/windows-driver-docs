# !amli cl


The **!amli cl** extension clears the AML interpreter's event log.

Syntax

```dbgcmd
    !amli cl
```

## <span id="ddk__amli_cl_dbg"></span><span id="DDK__AMLI_CL_DBG"></span>


### <span id="DLL"></span><span id="dll"></span>DLL

Kdexts.dll
