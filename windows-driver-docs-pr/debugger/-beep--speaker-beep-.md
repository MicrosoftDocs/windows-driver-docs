# .beep (Speaker Beep)


The **.beep** command makes noise on the computer speaker.

``` syntax
    .beep
```

### <span id="Environment"></span><span id="environment"></span><span id="ENVIRONMENT"></span>Environment

This command cannot be used in script files.
