# !usb3kd.dsf


The [**!usb3kd.dsf**](-usb3kd-device-info.md) extension is a toggle command that sets the debugger context to debug the DSF host driver.

```
!usb3kd.dsf
```

## <span id="DLL"></span><span id="dll"></span>DLL


Usb3kd.dll
