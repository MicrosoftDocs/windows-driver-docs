# .logclose (Close Log File)


The **.logclose** command closes any open log file.

```dbgcmd
    .logclose 
```

### Environment

|  Item  | Description          |
|--------|----------------------|
|Modes   |User mode, kernel mode|
|Targets |Live, crash dump      |
