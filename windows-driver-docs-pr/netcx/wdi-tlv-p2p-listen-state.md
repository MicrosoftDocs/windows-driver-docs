# WDI\_TLV\_P2P\_LISTEN\_STATE (dot11wificxtypes.hpp)

[!INCLUDE [WiFiCx topic note](../includes/wificx-version-warning.md)]


WDI\_TLV\_P2P\_LISTEN\_STATE is a TLV that contains a Wi-Fi Direct listen state.

## TLV Type


0x81

## Length


The sum (in bytes) of the sizes of all contained elements.

## Values


| Type                                                         | Description                            |
|--------------------------------------------------------------|----------------------------------------|
| [**WDI\_P2P\_LISTEN\_STATE**](/windows-hardware/drivers/ddi/dot11wificxtypes/ne-dot11wificxtypes-wdi_p2p_listen_state) | The desired Wi-Fi Direct listen state. |
