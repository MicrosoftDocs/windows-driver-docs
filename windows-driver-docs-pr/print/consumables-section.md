# Consumables Section


The Consumables section consists of the [Consumables](consumables3.md) property and a hierarchy of properties and values that provide information about the items consumed by the device.

```cpp
Printer
 Consumables
    [Name]
       Type
       Color
       Installed
       Level
       Model
```
