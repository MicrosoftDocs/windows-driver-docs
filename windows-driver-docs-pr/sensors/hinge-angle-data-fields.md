# Hinge angle sensor data fields

This topic provides information about the data fields that are specific to the hinge angle sensor.

The following table shows the data fields. For more information about the data types shown in the **Type** column, see the [PROPVARIANT structure](/windows/win32/api/propidlbase/ns-propidlbase-propvariant).

| Property key | Type | Required/Optional | Description |
|---|---|---|---|
| PKEY_SensorData_HingeAngle | VT_R4 | Required |  The interior hinge angle between two panels in a system, measured in degrees. |
