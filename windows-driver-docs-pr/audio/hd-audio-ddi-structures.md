# HD Audio DDI Structures


The routines in the two versions of the [HD Audio DDI](hd-audio-ddi-routines.md) use the following structure types:

[**HDAUDIO\_BUFFER\_DESCRIPTOR**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_buffer_descriptor)

[**HDAUDIO\_BUS\_INTERFACE**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface)

[**HDAUDIO\_BUS\_INTERFACE\_V2**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface_v2)

[**HDAUDIO\_BUS\_INTERFACE\_BDL**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_bus_interface_bdl)

[**HDAUDIO\_CODEC\_COMMAND**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_codec_command)

[**HDAUDIO\_CODEC\_RESPONSE**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_codec_response)

[**HDAUDIO\_CODEC\_TRANSFER**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_codec_transfer)

[**HDAUDIO\_CONVERTER\_FORMAT**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_converter_format)

[**HDAUDIO\_DEVICE\_INFORMATION**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_device_information)

[**HDAUDIO\_STREAM\_FORMAT**](/windows-hardware/drivers/ddi/hdaudio/ns-hdaudio-_hdaudio_stream_format)
