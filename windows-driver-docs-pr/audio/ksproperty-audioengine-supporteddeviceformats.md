# KSPROPERTY\_AUDIOENGINE\_SUPPORTEDDEVICEFORMATS


The **KSPROPERTY\_AUDIOENGINE\_SUPPORTEDDEVICEFORMATS** property request retrieves the device formats that are supported by the hardware audio engine.
