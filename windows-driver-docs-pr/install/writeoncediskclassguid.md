# WriteOnceDiskClassGuid


WriteOnceDiskClassGuid is an obsolete identifier for the [device interface class](https://msdn.microsoft.com/library/windows/hardware/ff541339) for write-once disk devices. Starting with Microsoft Windows 2000, use the [**GUID_DEVINTERFACE_WRITEONCEDISK**](guid-devinterface-writeoncedisk.md) class identifier for new instances of this class.
