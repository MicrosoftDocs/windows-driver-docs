# FsRtlAllocatePool

The **FsRtlAllocatePool** routine is obsolete, but is exported to support existing driver binaries. Use [**FsRtlAllocatePoolWithTag**](/windows-hardware/drivers/ddi/ntifs/nf-ntifs-fsrtlallocatepoolwithtag) instead.
