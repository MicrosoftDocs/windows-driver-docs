# FsRtlAllocatePool





The **FsRtlAllocatePool** routine is obsolete, but is exported to support existing driver binaries. Use [**FsRtlAllocatePoolWithTag**](https://msdn.microsoft.com/library/windows/hardware/ff545675) instead.
