# WIA\_IPS\_OPTICAL\_YRES


The WIA\_IPS\_OPTICAL\_YRES property contains the highest-supported vertical optical resolution of a device, in dots per inch (dpi). The WIA minidriver creates and maintains this property.

Property Type: VT\_I4

Valid Values: WIA\_PROP\_NONE

Access Rights: Read-only
