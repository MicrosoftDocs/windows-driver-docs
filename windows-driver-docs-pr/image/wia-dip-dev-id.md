# WIA\_DIP\_DEV\_ID


The WIA\_DIP\_DEV\_ID property contains the device identifier (ID) string for a WIA minidriver. The WIA service creates and maintains this property.

Property Type: VT\_BSTR

Valid Values: WIA\_PROP\_NONE
