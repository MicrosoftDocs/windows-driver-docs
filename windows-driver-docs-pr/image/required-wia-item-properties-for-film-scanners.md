# Required WIA Item Properties for Film Scanners





The WIA film scanner item is required to support the following WIA properties:

[**WIA\_IPA\_ACCESS\_RIGHTS**](./wia-ipa-access-rights.md)

[**WIA\_IPA\_BITS\_PER\_CHANNEL**](./wia-ipa-bits-per-channel.md)

[**WIA\_IPA\_COMPRESSION**](./wia-ipa-compression.md)

[**WIA\_IPA\_CHANNELS\_PER\_PIXEL**](./wia-ipa-channels-per-pixel.md)

[**WIA\_IPA\_DATATYPE**](./wia-ipa-datatype.md)

[**WIA\_IPA\_DEPTH**](./wia-ipa-depth.md)

[**WIA\_IPS\_FILM\_SCAN\_MODE**](./wia-ips-film-scan-mode.md)

[**WIA\_IPA\_FORMAT**](./wia-ipa-format.md)

[**WIA\_IPA\_FULL\_ITEM\_NAME**](./wia-ipa-full-item-name.md)

[**WIA\_IPA\_ITEM\_CATEGORY**](./wia-ipa-item-category.md)

[**WIA\_IPA\_ITEM\_NAME**](./wia-ipa-item-name.md)

[**WIA\_IPA\_ITEM\_SIZE**](./wia-ipa-item-size.md)

[**WIA\_IPA\_PREFERRED\_FORMAT**](./wia-ipa-preferred-format.md)

[**WIA\_IPA\_TYMED**](./wia-ipa-tymed.md)

[**WIA\_IPS\_BRIGHTNESS**](./wia-ips-brightness.md)

[**WIA\_IPS\_CONTRAST**](./wia-ips-contrast.md)

[**WIA\_IPS\_CUR\_INTENT**](./wia-ips-cur-intent.md)

[**WIA\_IPS\_MAX\_HORIZONTAL\_SIZE**](./wia-ips-max-horizontal-size.md)

[**WIA\_IPS\_MAX\_VERTICAL\_SIZE**](./wia-ips-max-vertical-size.md)

[**WIA\_IPS\_MIN\_HORIZONTAL\_SIZE**](./wia-ips-min-horizontal-size.md)

[**WIA\_IPS\_MIN\_VERTICAL\_SIZE**](./wia-ips-min-vertical-size.md)

[**WIA\_IPS\_OPTICAL\_XRES**](./wia-ips-optical-xres.md)

[**WIA\_IPS\_OPTICAL\_YRES**](./wia-ips-optical-yres.md)

[**WIA\_IPS\_PHOTOMETRIC\_INTERP**](./wia-ips-photometric-interp.md)

[**WIA\_IPS\_PREVIEW**](./wia-ips-preview.md)

[**WIA\_IPS\_XEXTENT**](./wia-ips-xextent.md)

[**WIA\_IPS\_XPOS**](./wia-ips-xpos.md)

[**WIA\_IPS\_XRES**](./wia-ips-xres.md)

[**WIA\_IPS\_YEXTENT**](./wia-ips-yextent.md)

[**WIA\_IPS\_YPOS**](./wia-ips-ypos.md)

[**WIA\_IPS\_YRES**](./wia-ips-yres.md)

**Note**   The [**WIA\_IPA\_RAW\_BITS\_PER\_CHANNEL**](./wia-ipa-raw-bits-per-channel.md) property is required if the WiaImgFmt\_RAW format is supported. The [**WIA\_IPA\_FORMAT**](./wia-ipa-format.md) property must support the WiaImgFmt\_BMP format. The [**WIA\_IPS\_THRESHOLD**](./wia-ips-threshold.md) property is required when the [**WIA\_IPA\_DEPTH**](./wia-ipa-depth.md) property is set to 1 bit per pixel (BPP) or when the [**WIA\_IPA\_DATATYPE**](./wia-ipa-datatype.md) property is set to WIA\_DATA\_THRESHOLD.
