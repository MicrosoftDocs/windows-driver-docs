# Web Services


The DSMdevice must support the following web services:

-   Web Services on Devices, including the Devices Profile for Web Services (DPWS).

-   The host side of the Distributed Scan Device Web Service protocol.

-   The client side of the Distributed Scan Processing Web Service protocol.

For more information about DPWS, see the [Devices Profile for Web Services](http://go.microsoft.com/fwlink/p/?linkid=59069) specification. For more information about the Distributed Scan Device Web Service and Distributed Scan Processing Web Service protocols, see [Distributed Scan Device Web Service Protocol Summary](distributed-scan-device-web-service-protocol-summary.md) and [Distributed Scan Processing Web Service Protocol Summary](distributed-scan-processing-web-service-protocol-summary.md).
