# Reset method of the MSFT\_StorageReliabilityCounter class

Resets reliability values.

## Syntax


```mof
UInt32 Reset();
```



## Parameters

This method has no parameters.

## Return value

 

**Success** (0)
 

**Not Supported** (1)
 

**Unspecified Error** (2)
 

**Timeout** (3)
 

**Failed** (4)
 

**Invalid Parameter** (5)
