# Clear method of the MSFT\_FileStorageTier class

Unpins a volume or file from a storage tier.

## Syntax


```mof
UInt32 Clear(
  [in] String FilePath
);
```



## Parameters

 

*FilePath* \[in\]
 

The path of a file to unpin.

| Requirement | Value |
