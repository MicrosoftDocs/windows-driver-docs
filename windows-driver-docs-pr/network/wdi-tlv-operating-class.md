# WDI\_TLV\_OPERATING\_CLASS


WDI\_TLV\_OPERATING\_CLASS is a TLV that contains the frequency band for a channel.

## TLV Type


0xFA

## Length


The size (in bytes) of a UINT8.

## Values


| Type  | Description                       |
|-------|-----------------------------------|
| UINT8 | The frequency band for a channel. |
