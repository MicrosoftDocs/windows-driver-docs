# OID_GEN_CO_XMIT_PDUS_ERROR

The OID_GEN_CO_XMIT_PDUS_ERROR OID specifies the number of PDUs a NIC failed to transmit.

## Requirements

| | |
| --- | --- |
| Version | Windows Vista and later |
| Header | Ntddndis.h (include Ndis.h) |
