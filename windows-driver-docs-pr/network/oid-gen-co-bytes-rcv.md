# OID_GEN_CO_BYTES_RCV

The OID_GEN_CO_BYTES_RCV OID specifies the number of bytes in PDUs received without errors.

## Requirements

| | |
| --- | --- |
| Version | Windows Vista and later |
| Header | Ntddndis.h (include Ndis.h) |
