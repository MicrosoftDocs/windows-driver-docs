# WDI_CHANNEL_NUMBER

The WDI_CHANNEL_NUMBER data type is a UINT32 value that defines a channel number.

```c++
typedef UINT32 WDI_CHANNEL_NUMBER;
```

## Requirements

|   |   |
| --- | --- |
| Minimum supported client | Windows 10 |
| Minimum supported server | Windows Server 2016 |
| Header | Wditypes.hpp |
