# WDI\_TLV\_AP\_CAPABILITIES


WDI\_TLV\_AP\_CAPABILITIES is a TLV that contains the capabilities of an access point.

## TLV Type


0x16

## Length


The sum (in bytes) of the sizes of all contained elements.
