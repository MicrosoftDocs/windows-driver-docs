# WDI\_TLV\_STATUS

[!INCLUDE [WDI topic note](../includes/wdi-version-warning.md)]


WDI\_TLV\_STATUS is a TLV that contains a status value.

## TLV Type


0x1

## Length


The size (in bytes) of an NDIS\_STATUS.

## Values


| Type         | Description             |
|--------------|-------------------------|
| NDIS\_STATUS | The NDIS\_STATUS value. |
