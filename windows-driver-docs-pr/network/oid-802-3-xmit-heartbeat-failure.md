# OID_802_3_XMIT_HEARTBEAT_FAILURE

The number of frames successfully transmitted without detection of the collision-detect heartbeat.

## Requirements

**Version**: Windows Vista and later
**Header**: Ntddndis.h (include Ndis.h)
