# WDI\_TLV\_IPV4\_LSO\_V2 (0xD3)


WDI\_TLV\_IPV4\_LSO\_V2 is a TLV that contains Large Send Offload V2 parameters for IPv4.

## TLV Type


0xD3

## Length


The sum (in bytes) of the sizes of all contained elements.
