# OID_GEN_CO_BYTES_XMIT

The OID_GEN_CO_BYTES_XMIT OID specifies the number of bytes in PDUs transmitted without errors.

## Requirements

**Version**: Windows Vista and later
**Header**: Ntddndis.h (include Ndis.h)
