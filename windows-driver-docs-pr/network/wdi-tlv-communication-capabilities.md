# WDI\_TLV\_COMMUNICATION\_CAPABILITIES


WDI\_TLV\_COMMUNICATION\_CAPABILITIES is a TLV that specifies the communication capabilities.

## TLV Type


0xE

## Length


The size (in bytes) of a UINT32.

## Values


| Type   | Description                        |
|--------|------------------------------------|
| UINT32 | The maximum command size in bytes. |
