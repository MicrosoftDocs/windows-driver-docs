# OID_TUNNEL_INTERFACE_SET_OID

The OID_TUNNEL_INTERFACE_SET_OID object identifier (OID) is reserved for system use. Do not use it in your driver.

## Requirements

**Version**: Windows 7 and later

**Header**: Ntddndis.h (include Ndis.h)
