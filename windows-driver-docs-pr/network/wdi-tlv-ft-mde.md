# WDI\_TLV\_FT\_MDE


WDI\_TLV\_FT\_MDE is a TLV that contains the MDIE of a BSS entry.

## TLV Type


0x10D

## Length


The size (in bytes) of the array of UINT8 elements. The array must contain 1 or more elements.

## Values


| Type      | Description              |
|-----------|--------------------------|
| UINT8\[\] | The MDIE of a BSS entry. |
