# WDI\_TLV\_P2P\_WPS\_ENABLED


WDI\_TLV\_P2P\_WPS\_ENABLED is a TLV that specifies if Wi-Fi Protected Setup is enabled.

## TLV Type


0xF7

## Length


The size (in bytes) of a UINT8.
