# WDI\_TLV\_CHANNEL\_NUMBER

[!INCLUDE [WDI topic note](../includes/wdi-version-warning.md)]


WDI\_TLV\_CHANNEL\_NUMBER is a TLV that contains a channel number.

## TLV Type


0x121

## Length


The size (in bytes) of a UINT8.

## Values


| Type  | Description         |
|-------|---------------------|
| UINT8 | The channel number. |
