# SRB\_PROPOSE\_STREAM\_RATE


## <span id="ddk_srb_propose_stream_rate_ks"></span><span id="DDK_SRB_PROPOSE_STREAM_RATE_KS"></span>


The **SRB\_PROPOSE\_STREAM\_RATE** command proposes a new rate but does not change it.
